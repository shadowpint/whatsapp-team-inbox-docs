## Step 1: Clone the Repository

```bash
git clone https://github.com/shadowpint/whatsapp-team-inbox.git
cd whatsapp-team-inbox
```

## Step 2: Backend Setup

<Steps>
  <Step title="Navigate to backend directory">
    ```bash
    cd backend
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    npm install
    ```
  </Step>

  <Step title="Create environment file">
    ```bash
    cp .env.example .env
    ```

    Edit `.env` with your configuration:

    ```bash
    # Database
    DATABASE_URL="postgresql://postgres:password@localhost:5432/whatsapp_inbox"

    # JWT Secret (generate with: node -e "console.log(require('crypto').randomBytes(64).toString('hex'))")
    JWT_SECRET="your-secret-key-here"

    # For development, you can skip WhatsApp and email setup initially
    ```
  </Step>

  <Step title="Setup database">
    ```bash
    npx prisma generate
    npx prisma db push
    ```
  </Step>

  <Step title="Start backend server">
    ```bash
    npm run dev
    ```

    You should see:
    ```
    ✓ Server running on http://localhost:4000
    ✓ Database connected
    ✓ WebSocket server running
    ```
  </Step>
</Steps>

## Step 3: Frontend Setup

Open a new terminal window:

<Steps>
  <Step title="Navigate to frontend directory">
    ```bash
    cd web
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    npm install
    ```
  </Step>

  <Step title="Create environment file">
    ```bash
    cp .env.local.example .env.local
    ```

    Edit `.env.local`:

    ```bash
    NEXT_PUBLIC_API_URL=http://localhost:4000
    NEXT_PUBLIC_WS_URL=ws://localhost:4000
    ```
  </Step>

  <Step title="Start frontend server">
    ```bash
    npm run dev
    ```

    You should see:
    ```
    ▲ Next.js 14.2.33
    - Local:        http://localhost:3000
    ✓ Ready in 2.5s
    ```
  </Step>
</Steps>

## Step 4: Create Your First User

<Steps>
  <Step title="Open your browser">
    Navigate to [http://localhost:3000](http://localhost:3000)
  </Step>

  <Step title="Register an account">
    - Click "Sign Up"
    - Enter your details
    - Complete registration
  </Step>

  <Step title="Login">
    - Use your credentials to log in
    - You'll be redirected to the inbox
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Database connection failed">
    - Ensure PostgreSQL is running: `pg_isready`
    - Check your `DATABASE_URL` in `.env`
    - Verify PostgreSQL is listening on port 5432
    - Try: `psql -U postgres` to test connection
  </Accordion>

  <Accordion title="Port already in use">
    - Backend (4000): `lsof -ti:4000 | xargs kill -9`
    - Frontend (3000): `lsof -ti:3000 | xargs kill -9`
    - Or change ports in `.env` and `.env.local`
  </Accordion>

  <Accordion title="Module not found errors">
    - Delete `node_modules` and reinstall:
      ```bash
      rm -rf node_modules package-lock.json
      npm install
      ```
  </Accordion>

  <Accordion title="Prisma errors">
    - Regenerate Prisma client:
      ```bash
      npx prisma generate
      npx prisma db push
      ```
  </Accordion>
</AccordionGroup>
