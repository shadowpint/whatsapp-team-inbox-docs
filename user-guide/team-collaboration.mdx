## Communication Guidelines

### Response Time Expectations

Set clear expectations:

| Mention Type | Expected Response | Example |
|--------------|------------------|---------|
| Urgent @mention | &lt;15 minutes | @john URGENT: VIP customer issue |
| Normal @mention | &lt;2 hours | @sarah Can you review this? |
| FYI mention | No response needed | @team FYI: System maintenance tonight |

### Handoff Protocol

Standard process for conversation transfers:

1. **Add context** in notes
2. **@Mention** new agent
3. **Wait for acknowledgment** (or escalate if urgent)
4. **Notify customer** about transfer
5. **Monitor** until new agent responds

## Team Roles

### Role-Based Permissions

Different access levels for team members:

<Tabs>
  <Tab title="Agent">
    **Can:**
    - View assigned conversations
    - Respond to customers
    - Add notes and tags
    - Search knowledge base
    - Update own profile

    **Cannot:**
    - View all conversations
    - Modify templates
    - Access admin settings
    - View reports
  </Tab>

  <Tab title="Team Lead">
    **Agent permissions plus:**
    - View team conversations
    - Reassign conversations
    - Create shared templates
    - View team metrics
    - Manage team schedule
    - Approve escalations
  </Tab>

  <Tab title="Manager">
    **Team Lead permissions plus:**
    - Access all conversations
    - Configure automations
    - Manage knowledge base
    - View full analytics
    - Export data
    - Manage integrations
  </Tab>

  <Tab title="Admin">
    **Full access:**
    - All manager permissions
    - User management
    - Billing access
    - Security settings
    - API configuration
    - System settings
  </Tab>
</Tabs>

## Notifications for Teams

Configure team-wide alerts:

### Notification Rules

**New conversation alerts:**
```javascript
{
  "trigger": "new_unassigned_conversation",
  "notify": ["team-lead@company.com"],
  "channel": "desktop",
  "conditions": {
    "tags": ["urgent", "vip"]
  }
}
```

**SLA breach warnings:**
```javascript
{
  "trigger": "sla_approaching",
  "notify": "assigned_agent",
  "escalate_to": "team_lead",
  "escalate_after_minutes": 15,
  "channel": ["desktop", "email"]
}
```
