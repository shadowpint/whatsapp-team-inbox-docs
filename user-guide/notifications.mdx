## Testing Notifications

Verify your notification settings:

<Steps>
  <Step title="Send Test Notification">
    ```
    Settings â†’ Notifications â†’ Test

    Send test:
    â˜‘ Desktop notification
    â˜‘ Email
    ```

  <Step title="Verify Receipt">
    Check each channel:
    - Desktop: Should see browser notification
    - Email: Check inbox (also spam folder)
    - Mobile: Check phone notification tray
  </Step>

  <Step title="Troubleshoot Issues">
    If not receiving:

    **Desktop:**
    - Check browser permissions
    - Verify notifications enabled in OS
    - Try different browser

    **Email:**
    - Check spam/junk folder
    - Verify email address correct
    - Check email provider not blocking

    **Mobile:**
    - Check app permissions
    - Verify notifications enabled in OS
    - Check Do Not Disturb mode
  </Step>
</Steps>
