## Welcome to Your Team Inbox

The WhatsApp Team Inbox is designed to help your team manage customer conversations efficiently. This guide will walk you through the main interface and key features.

<Info>
New to Team Inbox? Start here to learn the basics of navigating the interface and managing conversations.
</Info>

### Searching Conversations

The search bar helps you find conversations quickly:

<Steps>
  <Step title="Basic Search">
    ```
    Type customer name, phone number, or message content
    Results appear instantly as you type
    ```
  </Step>

  <Step title="Advanced Search">
    Use search operators for precise results:

    | Operator | Example | Description |
    |----------|---------|-------------|
    | `from:` | `from:john` | Messages from specific customer |
    | `tag:` | `tag:urgent` | Conversations with tag |
    | `status:` | `status:open` | Filter by status |
    | `assigned:` | `assigned:me` | Assigned conversations |
    | `has:` | `has:media` | Contains media |

    **Example:**
    ```
    tag:urgent status:open assigned:me
    ```
    Shows your urgent, open conversations.
  </Step>

  <Step title="Saved Searches">
    Save frequently used searches:
    1. Perform a search
    2. Click **Save Search**
    3. Name your search
    4. Access from **Saved** dropdown
  </Step>
</Steps>

### Composing Messages

<Steps>
  <Step title="Writing a Message">
    1. Click the message input box at the bottom
    2. Type your message
    3. Press **Enter** to send (or **Shift+Enter** for new line)
    4. Or click the send button â¤

    **Formatting options:**
    - **Bold**: `*text*` â†’ **text**
    - **Italic**: `_text_` â†’ *text*
    - **Strikethrough**: `~text~` â†’ ~~text~~
    - **Monospace**: `` `text` `` â†’ `text`
  </Step>

  <Step title="Adding Media">
    Click the attachment icon ğŸ“ to send:

    <Tabs>
      <Tab title="Images">
        - Supported: JPG, PNG, GIF
        - Max size: 5 MB
        - Add caption (optional)
        - Preview before sending

      <Tab title="Documents">
        - Supported: PDF, DOC, XLS, etc.

        - Max size: 16 MB

  <Step title="Using Quick Replies">
    Speed up responses with canned replies:

    1. Type `/` in the message box
    2. Browse or search templates

    **Example:**

    </Info>

### Conversation Management

<Tabs>
  <Tab title="Assignment">
    **Assign conversation to a team member:**

    1. Click **Assign** button
    2. Select team member
    3. Add note explaining assignment (optional)
    4. Click **Assign**

    The assigned person receives a notification.

    **Auto-assignment:**
    Enable to automatically distribute new conversations based on:
    - Round-robin
    - Current workload
    - Skill tags
    - Custom rules
  </Tab>

  <Tab title="Status">
    **Change conversation status:**

    - **Open**: Active, needs attention
    - **Pending**: Waiting for customer response
    - **Resolved**: Issue resolved, conversation closed

    Click status badge to change.

    **Auto-close:**
    Conversations automatically close after:
    - No response for X days (configurable)
    - Customer confirms resolution
    - Manual closure
  </Tab>

  <Tab title="Tags">
    **Organize with tags:**

    1. Click **+ Tag**
    2. Select existing tag or create new
    3. Apply multiple tags
    4. Use for filtering and reporting

    **Common tag examples:**
    - Priority: `urgent`, `normal`, `low`
    - Type: `sales`, `support`, `billing`
    - Status: `waiting-payment`, `escalated`
    - Product: `product-a`, `product-b`
  </Tab>

  <Tab title="Internal Notes">
    **Add private notes (not visible to customer):**

    1. Click **Notes** tab
    2. Write your note
    3. Mention team members with `@`
    4. Save note

    **Use cases:**
    - Document customer issues
    - Share context with team
    - Track action items
    - Log call summaries
  </Tab>
</Tabs>
